# Bubbletea TUI Development

Production-ready skill wrapper for building, reviewing, and migrating Bubble Tea TUI applications using the nested skill resources in `skills/bubbletea2`.

## When to use this skill

Use this skill when the task involves one or more of the following:

- Building a TUI with `Init/Update/View` architecture
- Adding Bubble Tea/Bubbles components to an existing app
- Fixing layout bugs, border rendering, overflow, or line wrapping
- Migrating Bubble Tea v1 code to v2
- Reviewing TUI PRs for behavioral regressions
